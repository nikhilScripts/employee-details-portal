MCP Configuration Guide for Cline AI + Jira + GitHub Integration

1. Jira API Token

Generate at: https://id.atlassian.com/manage-profile/security/api-tokens

Steps: Create API token → Copy (shown once only)

2. GitHub Personal Access Token

Generate at: GitHub → Settings → Developer Settings → Personal Access Tokens

Required Scopes: repo, workflow

3. MCP Settings File Location

%APPDATA%\Code\User\globalStorage\
saoudrizwan.claude-dev\settings\
cline_mcp_settings.json

4. Complete MCP Configuration (cline_mcp_settings.json)

{
  "mcpServers": {
    "jira": {
      "command": "node",
      "args": ["C:\\path\\to\\jira-full-server\\build\\index.js"],
      "env": {
        "JIRA_HOST": "https://YOUR-INSTANCE.atlassian.net",
        "JIRA_EMAIL": "your-email@example.com",
        "JIRA_API_TOKEN": "your-jira-api-token-here"
      },
      "disabled": false
    },
    "github": {
      "command": "cmd",
      "args": ["/c", "npx", "-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your-github-token-here"
      },
      "disabled": false
    }
  }
}

5. Available MCP Tools

Jira MCP Server Tools
get_issueGet issue by key (SCRUM-16)
create_issueCreate Story/Task/Bug under Epic
add_commentAdd closure comments
list_transitionsGet available status transitions
transition_issueMove to Done (ID: 41)
jql_searchSearch with JQL query
GitHub MCP Server Tools
create_repositoryCreate new repo
push_filesPush multiple files
create_branchCreate feature branch
create_pull_requestCreate PR
merge_pull_requestMerge PR
get_file_contentsRead file from repo

6. Workflow Summary

StepActionMCP Tool Used
1Give Epic ID (e.g., SCRUM-16)jira.get_issue
2AI creates child storiesjira.create_issue (with parentKey)
3AI generates code & docsCline write_to_file
4Create GitHub repogithub.create_repository
5Push all codegithub.push_files
6Add closure comments to storiesjira.add_comment
7Transition stories to Donejira.transition_issue
